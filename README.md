# A library for Futaba RS405CB servo motor.

## Usage

```shell
git clone https://github.com/satoshishimada/futaba_RS405CB
cd futaba_RS405CB
make
sudo chmod a+rw /dev/ttyUSB0
./serial [Servo ID]
```
